## 🚀 Déployer sur GitHub Pages

1. **Fork ou clone** ce repo
2. Va dans **Settings → Pages**
3. Dans *Source*, sélectionne **`main`** et **`/ (root)`**
4. Clique **Save**
5. Ton histoire est en ligne sur `https://TON-USERNAME.github.io/le-porteur-de-cendre/` en quelques secondes ✅
